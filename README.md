# Next.js + Headless WP Blog

A modern, headless WordPress blog built with Next.js, featuring dynamic article pages, responsive design, and seamless integration with WordPress REST API.

## Features

- Dynamic article routing with slug-based URLs
- Integration with WordPress CMS for content management
- TypeScript for type safety
- Tailwind CSS for styling

## Tech Stack

- **Frontend**: Next.js [v16.1.6], React [v19.2.3], TypeScript
- **Styling**: Tailwind CSS [v4]
